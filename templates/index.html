Overview

This is a data service that facilitates access to a variety of Alaska and Arctic geospatial data.

The code is hosted on Github. Instructions on how to use each data endpoint is given below.

Service endpoints

  • Wildfire
  • Permafrost
  • Integrated Ecosystem Model (IEM) data. This endpoint consolidates several variables (temperature, precipitation, and more).
  • Geology
  • Physiography
  • Glaciology
  • HUC Polygons
  • Forests and Vegetation
