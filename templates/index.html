Introduction

Welcome to SNAP’s data API, your gateway to seamless connectivity between our backend data and your front-end applications. With this versatile tool, you can access and manipulate the wealth of data we offer, and tailor it to suit your unique needs. You can also integrate the API into your own applications or scripts using the documentation outlined below for each of the endpoints offered. Whether you’re looking to highlight features of the data or to harness our data for custom analyses, this API is your one-stop shop for all of SNAP’s data offerings. For further details and inquiries, don’t hesitate to reach out to us at uaf-snap-data-tools@alaska.edu. We’re here to make climate data easier to access and understand!

The code is hosted on GitHub. Instructions on how to use each data endpoint is given below.

Rasdaman raster database

This tool uses the rasdaman array database management system. This technology enables our team to store, analyze and visualize the 5+ TB of climate data that are presented through this interface. We wish to thank the rasdaman project and team for their inspiration and support, and for providing this important innovation in data technology.

Peter Baumann, email: p.baumann@jacobs-university.de, & website: rasdaman.org. (2018). rasdaman - raster data manager (9.5.0). Zenodo. https://doi.org/10.5281/zenodo.1163021
