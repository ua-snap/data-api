ALFRESCO point query

Query data derived from 1km ALFRESCO for a single point specified by latitude and longitude.

Examples for the currently available datasets:

  • Relative flammability: /alfresco/flammability/point/65.0628/-146.1627
  • Relative vegetation change: /alfresco/veg_change/point/65.0628/-146.1627

CSV Export

To export the aggregated HUC data in a CSV file, append ?format=csv.

Example: /alfresco/flammability/huc/19080309?format=csv
