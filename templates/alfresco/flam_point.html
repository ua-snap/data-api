Relative flammability point query

Query ALFRESCO relative flammability data for a single point specified by latitude and longitude.

Example: /alfresco/flammability/point/65.0628/-146.1627

Results from the queries above will look like this:

{
  "1950-2008": {
    "CRU-TS40": {
      "CRU_historical": {
        "rf": 0.0025
      }
    }
  }
...
  "2010-2019": {
    "5modelAvg": {
      "rcp45": {
        "rf": 0.0011
      }
    }
  }
...
}
The above output is structured as such: 
{
  <era>: {
    <model>: {
      <scenario>: {
        <varname>: {
          <varname>: <value>
        }
      }
    }
  } 
  ...
}


Dimensions:

  • era: Sumary era:
    • 1950-1959
    • 1950-2008
    • 1960-1969
    • 1970-1979
    • 1980-1989
    • 1990-1999
    • 2000-2008
    • 2040_2069
    • 2070_2099
    • 2010_2019
    • 2020_2029
    • 2030_2039
    • 2040_2049
    • 2050_2059
    • 2060_2069
    • 2070_2079
    • 2080_2089
    • 2090_2099
  • model: AR5 model source, or CRU TS:
    • NCAR-CCSM4
    • MRI-CGCM3
    • IPSL-CM5A-LR
    • GFDL-CM3
    • GISS-E2-R
    • 5modelAvg: the average of the above 5 models
  • scenario: RCP scenario for AR5, or "CRU_historical" for CRU TS
    • rcp45
    • rcp60
    • rcp85
    • CRU_historical
Variables:
  • rf: (float), relative flammability.

Source data

Relative flammability is the average number of times a pixel burned over all reps for all years in the summary time period. These data are derived from ALFRESCO model outputs created for the IEM project. For access to the source data used here, please contact uaf-snap-data-tool@alaska.edu and ask about accessing these data (mention this page). SNAP has a similar dataset, "ALFRESCO Model outputs - Relative Flammability" (link to downloads: http://ckan.snap.uaf.edu/dataset/alfresco-model-outputs-relative-flammability/resource/07b4a7c2-36a5-41d0-99ba-2fc0c336d26e/), which provides grids of these data for coarser time periods and a subset of the models. {% endblock %}
