ALFRESCO Local aggregation

Query data derived from 1km ALFRESCO outputs for a point location specified by latitude and longitude, and return the mean of the pixels within the HUC-12 that intersects the supplied point location.

Examples for the currently available datasets:

  • Relative flammability: /alfresco/flammability/local/65.4844/-145.4036
  • Relative vegetation change: /alfresco/veg_change/local/65.4844/-145.4036
  • Vegetation type: /alfresco/veg_type/local/65.4844/-145.4036

CSV Export

To export the local data in a CSV file, append ?format=csv.

Example: /alfresco/flammability/local/65.4844/-145.4036?format=csv

12-digit HUC shapefiles accessed at https://datagateway.nrcs.usda.gov/
