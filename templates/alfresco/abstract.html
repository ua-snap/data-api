Post-processed ALFRESCO Data

The endpoints here provide access to data products derived from outputs from ALFRESCO simulations done using future (AR5/CMIP5) scenarios and historical (CRU TS 4.0) data completed for the Integrated Ecosystem Model (IEM) project. The current endpoints are described in the "Service Endpoints" section below, and are available for the following datasets:

Datasets

Endpoint Name Variable Name Data Layer Description
flammability rf Relative Flammability These data are derived from the fire scar outputs and are computed as the average number of times a pixel burned across all reps for all years of the summary year span.
veg_change rvc Relative Vegetation Change These data are derived from the vegetation class data, the principal phenomena being modeled by ALFRESCO. They are computed as the average number of year-to-year transitions of any vegetation class for a pixel, across all reps for all years of the summary year span.

Results from queries to the above endpoints will look like this:

{
  "1950-1959": {
    "CRU-TS40": {
      "CRU_historical": {
        "rf": 0.0009
      }
    }
  }
...
  "2010-2019": {
    "5modelAvg": {
      "rcp45": {
        "rf": 0.0017
      }
    }
  }
...
}
The above output is structured as such: 
{
  <era>: {
    <model>: {
      <scenario>: {
        <varname>: <value>
      }
    }
  } 
  ...
}

Dimensions:

  • era: Sumary era:
    • 1950-1959
    • 1950-2008
    • 1960-1969
    • 1970-1979
    • 1980-1989
    • 1990-1999
    • 2000-2008
    • 2040_2069
    • 2070_2099
    • 2010_2019
    • 2020_2029
    • 2030_2039
    • 2040_2049
    • 2050_2059
    • 2060_2069
    • 2070_2079
    • 2080_2089
    • 2090_2099
  • model: AR5 model source, or CRU TS:
    • NCAR-CCSM4
    • MRI-CGCM3
    • IPSL-CM5A-LR
    • GFDL-CM3
    • GISS-E2-R
    • 5modelAvg: the average of the above 5 models
  • scenario: RCP scenario for AR5, or "CRU_historical" for historical
    • rcp45
    • rcp60
    • rcp85
    • CRU_historical
Variables:
  • rf: (float), relative flammability.
  • rvc: (float), relative vegetation change.

Source data

These data are derived from ALFRESCO model outputs created for the IEM project. For access to the source data used here, please contact uaf-snap-data-tool@alaska.edu and ask about accessing these data. SNAP also has some similar datasets available for download at longer year spans and for fewer models and scenarios than available here:
  • ALFRESCO Model outputs - Relative Vegetation Change
  • ALFRESCO Model outputs - Relative Flammability
