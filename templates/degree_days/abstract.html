Degree Days

The endpoints here provide access to the total annual cumulative degree days for several different degree day thresholds.

Datasets

Endpoint Threshold
heating degree days (below 65°F)
below_zero degree days below 0°F
thawing_index degree days above freezing (above 32°F)
below_zero degree days below freezing (below 32°F)

Examples

Heating Degree Days

Historical: /degree_days/heating/historical/65.0628/-146.1627

Projected: /degree_days/heating/projected/65.0628/-146.1627

Historical + projected: /degree_days/heating/hp/65.0628/-146.1627

Custom year range: /degree_days/heating/hp/65.0628/-146.1627/1990/2030

All: /degree_days/heating/all/65.0628/-146.1627

Output

Min/Mean/Max

Output from the historical, projected, or hp endpoints will look similar to this:

{
  "historical": {
    "max": 14972,
    "mean": 13780.2,
    "min": 9357
  },
  "projected": {
    "max": 15921,
    "mean": 12053.6,
    "min": 8036
  }
}

CSV Export

CSV data can be exported from any of the all endpoints by appending ?format=csv to the end of the URL.

All heating degreee day data (CSV): /degree_days/heating/all/65.0628/-146.1627?format=csv

All below 0°F degree day data (CSV): /degree_days/below_zero/all/65.0628/-146.1627?format=csv

All thawing index data (CSV): /degree_days/thawing_index/all/65.0628/-146.1627?format=csv

All freezing index data (CSV): /degree_days/freezing_index/all/65.0628/-146.1627?format=csv
