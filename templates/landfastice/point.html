Landfast Sea Ice Extent Point Query

This endpoint accesses daily landfast ice extent information for the Alaska Coastline between October 17th, 1996 and July 14th, 2008.

Fetch the entire daily time series

Query the daily landfast ice extent for a certain geographic point:

Example: /landfastice/point/70.6123/−149.8869

Results from queries like the above example will look like this: 
  {
    "01-01-1997": 0, 
    "01-01-1998": 0, 
    "01-01-1999": 0, 
    "01-01-2000": 1, 
    "01-01-2001": 1, 
    "01-01-2002": 1, 
    "01-01-2003": 1, 
    "01-01-2004": 1, 
    "01-01-2005": 1, 
    "01-01-2006": 1, 
    "01-01-2007": 1, 
    "01-01-2008": 0,
    ... 
}
The above output is structured like this: 
{
    "<month>–<day>–<year>": <landfast ice presence (1) or absence (0) for a specific date in the time series>
    ...
}
