Integrated Ecosystem Model (IEM) data

These data are for use in the IEM webapp. Currently, only SNAP's 2km downscaled temperature and precip data are available.

Service endpoints

  • Point Query: 2km temperature and precip data at a single point
  • Aggregate by HUC: 2km temperature and precip data within a HUC-8 polygon, aggregating by mean

Results from the queries above will look like this:

{
  "1910-2019": {
    "DJF":{
      "CRU-TS40": {
        "CRU_historical": {
          "pr": 39,
          "tas": -19.3
        }
      }
    }
  } 
  ...
}
The above output is structured as such: 
{
  <period>: {
    <season>:{
      <model>: {
        <scenario>: {
          "pr": <precip value>,
          "tas": <termperature value>
        }
      }
    }
  } 
  ...
}

Dimensions:

  • period: Sumary period:
    • 1950_2009
    • 2040_2069
    • 2070_2099
    • 2010_2019
    • 2020_2029
    • 2030_2039
    • 2040_2049
    • 2050_2059
    • 2060_2069
    • 2070_2079
    • 2080_2089
    • 2090_2099
  • season: Summary season:
    • DJF
    • MMA
    • JJA
    • SON
  • model: AR5 model source, or CRU TS:
    • CCSM4
    • MRI-CGCM3
    • CRU-TS40
  • scenario: RCP scenario for AR5, or "CRU_historical" for CRU TS
    • rcp45
    • rcp60
    • rcp85
    • CRU_historical
Variables::
  • pr: (float), Precipitation value.
  • tas: (float), Temperature value.
