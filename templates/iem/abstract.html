Integrated Ecosystem Model (IEM) data

These data are for use in the IEM webapp. Currently, only SNAP's 771m downscaled temperature and precip data are available.

Service endpoints

  • Point Query: 771m temperature and precip data at a single point
  • Aggregate by HUC: 771m temperature and precip data within a HUC-8 polygon, aggregating by mean

Results from the queries above will look like this:

{
  "1910-2009": {
    "DJF":{
      "CRU-TS31": {
        "CRU_historical": {
          "pr": 39.4,
          "tas": -19.3
        }
      }
    }
  } 
  ...
}
The above output is structured as such: 
{
  <period>: {
    <season>:{
      <model>: {
        <scenario>: {
          "pr": <precip value>,
          "tas": <termperature value>
        }
      }
    }
  } 
  ...
}

Dimensions:

  • period: Sumary period:
    • 1910-2009
    • 2040-2070
    • 2070-2100
  • season: Summary season:
    • DJF
    • MMA
    • JJA
    • SON
  • model: AR5 model source, or CRU TS:
    • NCAR-CCSM4
    • MRI-CGCM3
    • CRU TS v3.1
  • scenario: RCP scenario for AR5, or "CRU_historical" for CRU TS
    • rcp45
    • rcp60
    • rcp85
    • CRU_historical
Variables::
  • pr: (float), Precipitation value.
  • tas: (float), Temperature value.
