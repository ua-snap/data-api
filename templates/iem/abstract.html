Data API

IEM Data

These data are for use in the IEM Webapp. Currently, only SNAP's 771m downscaled temperature and precip data are available.

Service endpoints

  • Point Query
  • Aggregate by HUC

/iem/point

Query 771m temperature and precip data at a single point.

/iem/huc

Query 771m temperature and precip data within a HUC-8 polygon, aggregating by mean.
