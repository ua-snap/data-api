Temperature and Precipitation Point Query

Query SNAP's 2km downscaled temperature or precipitation data for a single point specified by latitude and longitude.

Example: /temperature/point/65.0628/-146.1627

Example: /precipitation/point/65.0628/-146.1627

Query both temperature and precipitation data for a single point: /taspr/point/65.0628/-146.1627

CSV Export

To export the point data in a CSV file, append ?format=csv.

Example: /temperature/point/65.0628/-146.1627?format=csv

Example: /taspr/point/65.0628/-146.1627?format=csv
