IEM Point Query

Query the 2km downscaled temperature and precipitation data for a single point specified by latitude and longitude.

Example: /iem/point/65.0628/-146.1627

CSV Export

To export the point data in a CSV file, append ?format=csv.

Example: /iem/point/65.0628/-146.1627?format=csv
