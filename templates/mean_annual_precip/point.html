Mean Annual Precipitation point query

Query mean annual precipitation projections for a single point specified by latitude and longitude.

Example: /mean_annual_precip/point/65.0628/-146.1627

Results from the query above will look like this:

{
  "dec_mapr": [
    {
      "dec_end": "2059",
      "dec_start": "2050",
      "model": "5modelAvg",
      "pr_mm": 408,
      "scenario": "rcp60",
      "title": "Mean Annual Precipitation (mm) Decadal Summary Projection for 2050-2059"
    },
    {
      "dec_end": "2059",
      "dec_start": "2050",
      "model": "5modelAvg",
      "pr_mm": 436,
      "scenario": "rcp85",
      "title": "Mean Annual Precipitation (mm) Decadal Summary Projection for 2050-2059"
    }
  ]
}
  • dec_mapr: Decadal Summaries of Mean Annual Precipitation Projections
    • dec_end: (str), End of decade calendar year.
    • dec_start: (str), Start of decade calendar year.
    • model: (str), Climate model used to generate outputs.
    • scenario: (str), RCP used to generate outputs.
    • pr_mm: (float), Mean Annual Precipitation (mm).
    • title: (str), Title describing these results.
