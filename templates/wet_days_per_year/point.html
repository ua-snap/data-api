Wet Days Per Year Point Query

The endpoints here provide access to wet days per year data.

Endpoints

Endpoint Description
wet_days_per_year The number of days per calendar year where total precipitation amounts are greater than or equal to 1.0 mm.

Examples

Wet Days Per Year

Query the downscaled ERA-Interim historical average (1980-2009) min, mean, and max number of wet days per year for a point location:

Example: /wet_days_per_year/historical/point/65.0628/-146.1627

Query the projected RCP 8.5 mid-century (2040-2069) two-model min, mean, and max number of wet days per year for a point location:

Example: /wet_days_per_year/projected/point/65.0628/-146.1627

Query for the historical and projected min, mean, and max numbers of wet days per year for a point location:

Example: /wet_days_per_year/hp/point/65.0628/-146.1627

Results from the queries above will look like this: 
{
  "historical": {
    "wdpymax": 160, 
    "wdpymean": 134, 
    "wdpymin": 112
  }
}
The above output is structured as such: 
{
  <era>: {
    "wdpymax": <max number of wet days per year across all models and years used for the summary era time period>,
    "wdpymean": <number of wet days per year across all models and years used for the summary era time period>,
    "wdpymin": <number of wet days per year across all models and years used for the summary era time period>
  },
  ...
}

Comprehensive queries

The following query will fetch a complete wet days per year set of data composed of the ERA-Interim historical baseline (1980-2009) and the entire projection range (2006-2100) for both available models, GFDL-CM3 and NCAR-CCSM4 (both RCP 8.5).

Example: /wet_days_per_year/all/point/65.0628/-146.1627

Results from the query above will look like this: 
{
  "ERA-Interim": {
    "1980": {
      "wdpy": 124
    }, 
    "1981": {
      "wdpy": 119
    }, 
    "1982": {
      "wdpy": 145
    }, 
...
}
The above output is structured as such: 
{
  <model>: {
    <year>: {
      "wdpy": <count of wet days per year>
    },
    ...
  }
  ...
}

CSV Export

To export the point data to a CSV file, append ?format=csv.

Example: /wet_days_per_year/all/point/65.0628/-146.1627?format=csv
