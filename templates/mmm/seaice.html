Pan Arctic Sea Ice Concentration Min, Mean, and Max

This endpoint accesses historical monthly data on sea ice concentration in the Arctic from 1850-2021 and returns minimum, mean, and maximum values.

Historical queries for min, mean, and max at given latitude / longitude

Query the historical sea ice concentration at lat / lon:

Example: /mmm/seaice/63.73/-166.32

Results from the query above will look like this: 
{
  "historical": {
    "seaice_max": 100,
    "seaice_mean": 49,
    "seaice_min": 0
  }
}

Historical queries for min, mean, and max at given latitude / longitude between provided years

Query the historical sea ice concentration at lat / lon between years:

Example: /mmm/seaice/63.73/-166.32/1900/2000

Results from the query above will look like this: 
{
  "historical": {
    "seaice_max": 100,
    "seaice_mean": 50,
    "seaice_min": 0
  }
}
