Annual Total Precipitation Min, Mean, and Max Point Query

Query for historical annual total precipitation min, mean, and max:

Example: /mmm/precipitation/historical/65.0628/-146.1627

Query for projected annual total precipitation min, mean, and max:

Example: /mmm/precipitation/projected/65.0628/-146.1627

Query for both historical and projected annual total precipitation min, mean, and max:

Example: /mmm/precipitation/hp/65.0628/-146.1627

Query for a given date range between 1901-2100 for min, mean, and max to be determined from.

Add a start year and end year to end of query:

Example: /mmm/precipitation/hp/65.0628/-146.1627/1901/2020

Query for all annual total precipitation data across all historical and projected models from 1900-2100:

Example: /mmm/precipitation/all/65.0628/-146.1627

Results from the query above will look like this: 
{
  "CRU-TS": {
    "historical": {
      "1900": {
        "pr": 140
      }
  }
  ...
}
The above output is structured as such: 
{
  <model>: {
    <scenario>:{
      <year>: {
        "pr": <pr value>
        }
      }
    }
  }
  ...
}

CSV Export

To export the annual total precipitation point data in a CSV file, append ?format=csv.

Example: /mmm/precipitation/all/65.0628/-146.1627?format=csv
