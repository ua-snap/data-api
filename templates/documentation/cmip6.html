CMIP6

These endpoints provide access to monthly data for select CMIP6 variables, models, and scenarios. Read more about our CMIP6 subset here. This dataset covers years 1950–2100 and is pan-Arctic in extent.

For more detailed information, see the links to academic references and source datasets included at the bottom of this page.

Service endpoints

CMIP6 monthly data

Query CMIP6 variables for all models and scenarios. These variables are defined below.

Endpoint Example URL
Monthly values at a given latitude and longitude (all years). /cmip6/point/61.5/-147
Monthly values at a given latitude and longitude (select years). /cmip6/point/61.5/-147/2030/2050
The list of variables can be limited by appending ?vars=<var_id> to the URL.
Multiple variables can be requested by appending a comma-separated string like ?vars=<var_id_1>,<var_id_2>.
CSV output is also available by appending ?format=csv to the URL.
Chaining of multiple arguments is supported (e.g., ?vars=pr,tas&format=csv).

CMIP6 Variable Information

Below is the list of CMIP6 variables that are found in this endpoint. For complete variable definitions, see the official CMIP6 documentation here.

clt
Cloud area fraction (%)
evspsbl
Evaporation, including sublimation and transpiration (kg m-2 s-1)
hfls
Surface upward latent heat flux (W m-2)
hfss
Surface upward sensible heat flux (W m-2)
pr
Precipitation, including both liquid and solid phases (kg m-2 s-1)
prsn
Precipitation as snow, including all forms of water in the solid phase (kg m-2 s-1)
psl
Sea level pressure (Pa)
rlds
Surface downwelling longwave radiation (W m-2)
rsds
Surface downwelling shortwave radiation (W m-2)
sfcWind
Surface wind speed (m s-1)
siconc
Sea ice concentration (%)
swe
Snow water equivalent (mm)
tas
Near surface air temperature (°C)
tasmax
Maximum near surface air temperature (°C)
tasmin
Minimum near surface air temperature (°C)
ts
Surface air temperature (°C)
uas
Near-surface eastward wind (m s-1)
vas
Near-surface northward wind (m s-1)
