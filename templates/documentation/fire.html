Wildfire Data

This endpoint provides access to near-real-time, historical, and model output relevant to Alaska wildfires. Daily fire danger classifications were provided by MesoWest Alaska Fires & Fuels at a resolution of 2.9km. Daily snow cover status is provided by the U.S. National Ice Center's Interactive Multisensor Snow and Ice Mapping System (IMS) at a resolution of 994m. Land cover types from 2015 were provided by the North American Land Change Monitoring System at a resolution of 30m. Historical fires from 1940–2021 were provided by the Alaska Interagency Coordination Center (AICC). Flammability projections are summarized across years 2000–2099 and were provided by ALFRESCO model outputs, simulated with the CRU TS 4.0 dataset (2000–2015) and NCAR-CCSM4 model outputs using the RCP 8.5 emissions scenario (2016–2099).

Links to academic references and source datasets are included at the bottom of this page.

Output

Results from wildfire queries will look like this:

{
  "cfd": {
    "code": 1,
    "color": "#2b83ba",
    "title": "Today's Fire Danger",
    "type": "Low"
  },
  "hist_fire": {
    "Angel Creek - Chena Dome": "1950",
    "WEST FORK CHENA": "2002"
  },
  "is_snow": {
    "is_snow": false,
    "title": "Today's Snow Cover"
  },
  "lc": {
    "code": 8,
    "color": "#b38a33",
    "title": "Land cover types",
    "type": "Temperate or sub-polar shrubland"
  },
  "prf": {
    "flamm": 0.0057,
    "title": "Projected relative flammability"
  }
}

The above output is structured like this:

{
  "cfd": {
    "code": <numeric fire danger rating, range 1 through 5>,
    "color": <hex color code to visualize fire danger>,
    "title": <title describing these data>,
    "type": <fire danger rating>
  },
  "hist_fire": {
    <historical fire name>: <fire year>,
    ...
  },
  "is_snow": {
    "is_snow": <true if point is covered by snow, otherwise false>,
    "title": <title describing these data>
  },
  "lc": {
    "code": <numeric land cover code>,
    "color": <hex color code to visualize land cover>,
    "title": <title describing these data>,
    "type": <Land cover description>
  },
  "prf": {
    "flamm": <numeric flammability value>,
    "title": <title describing these data>
  }
}
