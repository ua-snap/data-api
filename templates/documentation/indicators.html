Climate Indicators

These endpoints provide access to era-based summaries of the NCAR 12km climate indicators. The eras for this dataset are historical (1988–2017), mid–century (2040–2069), and late–century (2070–2099).

For more detailed information, see the links to academic references and source datasets included at the bottom of this page.

Service endpoints

All climate indicator variables (point query)

Query min-mean-max climate indicator variables for all models and scenarios.

Endpoint Example URL
Era-based min-mean-max for all climate indicator variables at a given latitude and longitude. /indicators/base/point/61.5/-147
Era-based min-mean-max for all climate indicator variables for a specific area of interest polygon ID and aggregate by mean. /indicators/base/area/61.5/-147
CSV output is also available by appending ?format=csv to the URL.

The above output is structured like this:

{
  <variable>: {
    <era>: {
      <model>: {
        <scenario>: {
          "max": <monthly sum of daily evapotranspiration, mm>,

        },
        ...
      },
      ...
    },
    ...
  },
  ...
}
