Post-processed 1km ALFRESCO Data

These endpoints provide access to flammability and vegetation data products derived from ALFRESCO simulations at a resolution of 1km. Historical data are derived from the CRU TS 4.0 dataset and are summarized across years 1950–1979 and 1980–2008. Projections were simulated using the GFDL-CM3, GISS-E2-R, IPSL-CM5A-LR, MRI-CGCM3, and NCAR-CCSM4 model outputs using the RCP 4.5, RCP 6.0, and RCP 8.5 emissions scenarios. Projections simulating a 5-model average using RCP 4.5, RCP 6.0, and RCP 8.5 are also available. Projections are provided as summarized means of 30-year eras from 2010–2099.

See the ALFRESCO and Habitat Research project page for more information.

Datasets

Dataset Description
Flammability These data are derived from the fire scar outputs and are computed as the average number of times a pixel burned across all reps for all years of the summary year span. Pixels are 1km x 1km.
Vegetation Type These data are derived from the vegetation class data, the principal phenomena being modeled by ALFRESCO. They are computed as the average percentage of each vegetation type for a pixel, across all reps for all years of the summary year span. Pixels are 1km x 1km.

Service endpoints

Point query

Query data derived from 1km ALFRESCO for a single point specified by latitude and longitude.

Endpoint Example URL
Flammability point query /alfresco/flammability/point/65.0628/-146.1627
Vegetation type point query /alfresco/veg_type/point/65.0628/-146.1627
CSV output is also available by appending ?format=csv to the URL.

Intersecting area query

Query data derived from 1km ALFRESCO outputs for a point location specified by latitude and longitude, and return the mean of the pixels within the HUC-12 that intersects the supplied point location. 12-digit HUC shapefiles accessed at https://datagateway.nrcs.usda.gov/

Endpoint Example URL
Flammability intersecting area query /alfresco/flammability/local/65.4844/-145.4036
Vegetation type intersecting area query /alfresco/veg_type/local/65.4844/-145.4036
CSV output is also available by appending ?format=csv to the URL.

Area query

Query data derived from 1km ALFRESCO outputs for an area of interest polygon ID and aggregate by mean. 8-digit HUC shapefiles accessed at https://datagateway.nrcs.usda.gov/

Endpoint Example URL
Flammability area query /alfresco/flammability/area/19080309
Vegetation type area query /alfresco/veg_type/area/19080309
CSV output is also available by appending ?format=csv to the URL.

Output

Flammability

Results from flammability endpoints will look like this:

{
  "1950-1979": {
    "MODEL-SPINUP": {
      "historical": 0.0065
    }
  },
  ...
  "2010-2039": {
    "5modelAvg": {
      "rcp45": 0.0014,
      "rcp60": 0.0007,
      "rcp85": 0.0007
    },
    ...
  },
  ...
}

The above output is structured like this:

{
  <era>: {
    <model>: {
      <scenario>: <flammability>
    },
    ...
  },
  ...
}

Vegetation type

{
  "1950-2008": {
    "MODEL-SPINUP": {
      "historical": {

        "not_modeled": 0,
        "shrub_tundra": 0,
        "temperate_rainforest": 0,
        "wetland_tundra": 0,
        "white_spruce": 49.08
      }
    }
  },
  ...
  "NCAR-CCSM4": {
    "rcp45": {
      "barren_lichen_moss": 0,
      "black_spruce": 0,
      "deciduous_forest": 57.03,
      "graminoid_tundra": 0,
      "not_modeled": 0,
      "shrub_tundra": 0,
      "temperate_rainforest": 0,
      "wetland_tundra": 0,

    ...
  },
  ...

The above output is structured like this:

{
  <era>: {
    <model>: {
      <scenario>: {
        <vegetation type>: <percentage>
      },
      ...
    },
    ...
  },
  ...
}
