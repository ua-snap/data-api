Geology Data

These data are for use in the Arctic-EDS. Currently, only the USGS detailed rock unit names and ages are available.

Service endpoints

Point query

Query the USGS Geologic Map of Alaska at a single point specified by latitude and longitude.

Endpoint Example URL
Geology point query /geology/point/69.0628/-144.1627

Output

Results from point queries will look like this:

{
  "age": "Cambrian and Proterozoic",
  "name": "Wickersham and Neruokpuk units",
  "title": "USGS Geologic Map of Alaska"
}

The above output is structured like this:

{
  "age": <geologic unit age>,
  "name": <geologic unit name>,
  "title": <title describing these results>,
}
