Permafrost Data

These endpoints provide access to permafrost information for Alaska, including outputs from academic research on permafrost type and extent as well as ground temperatures.

Annual projections of permafrost top and base depths, talik thickness, and mean annual ground temperature at seven different depths were provided by the GIPL 2.0 model at a resolution of 1km for years 2021–2120. These projections are provided for GFDL-CM3 and NCAR-CCSM4 model outputs, as well as a 5-model average, using both the RCP 4.5 and RCP 8.5 emissions scenarios.

Active layer thickness and mean annual ground temperature data are also provided as summarized means across 2011–2040, 2036–2065, 2061–2090, 2086–2100 vs. a historical baseline from 1995 at a resolution of 4km. Data were provided by Melvin et al. using GIPL 2.0 model outputs. The historical baseline was derived from CRU TS 3.1. Projections were derived from GFDL-CM3, GISS-E2R, IPSL-CM5A-LR, MRI-CGCM3, NCAR-CCSM4 model outputs, as well as a 5-model average, using both the RCP 4.5 and RCP 8.5 emissions scenarios.

Permafrost extent and ground ice volume data from 2008 were provided by Jorgenson et al. Mean annual top of permafrost ground temperature and permafrost extent data from 2000–2016 were provided by Obu et al.

Service endpoints

Geophysical Institute Permafrost Lab (GIPL) model output (point query)

Query GIPL 2.0 model outputs (1 km spatial resolution) for ten variables for a single point specified by latitude and longitude.

Endpoint Example URL
GIPL point query of entire time series /permafrost/point/gipl/63.0628/-146.1627
GIPL point query within year range /permafrost/point/gipl/63.0628/-146.1627/2040/2050
CSV output is also available by appending ?format=csv to the URL.

Geophysical Institute Permafrost Lab (GIPL) min/mean/max summaries (point query)

Query minimum, mean, and maximum summaries of GIPL 2.0 model outputs (1 km spatial resolution) for ten variables for a single point specified by latitude and longitude.

Endpoint Example URL
GIPL point query within year range min/mean/max summary /permafrost/point/gipl/63.0628/-146.1627/2040/2050/mmm
CSV output is also available by appending ?format=csv to the URL.

Permafrost model output summaries from multiple sources (point query)

Query permafrost model outputs (e.g. extent, ground temperatures) from multiple sources.

Endpoint Example URL
Multiple permafrost data sources point query summary /permafrost/point/65.0628/-146.1627
CSV output is also available by appending ?format=csv to the URL.

Output

Geophysical Institute Permafrost Lab (GIPL) model output

Results from GIPL entire series or time slice queries will look like this:

{
  "5ModelAvg": {
    "2021": {
      "RCP 4.5": {
        "magt0.5m": -2.6,
        "magt1m": -3.1,
        "magt2m": -3.3,
        "magt3m": -3.4,
        "magt4m": -3.5,
        "magt5m": -3.5,
        "magtsurface": -0.0,
        "permafrostbase": 153.2,
        "permafrosttop": 0.7,
        "talikthickness": 0
      },
      ...
    },
    ...
  },
  ...
}

The above output is structured like this:

{
  <model>: {
    <year>:{
      <scenario>: {
        "magt0.5m": <mean annual ground temperature value at 0.5 m depth (°C)>,
        "magt1m": <mean annual ground temperature value at 1 m depth (°C)>,
        "magt2m": <mean annual ground temperature value at 2 m depth (°C)>,
        "magt3m": <mean annual ground temperature value at 3 m depth (°C)>,
        "magt4m": <mean annual ground temperature value at 4 m depth (°C)>,
        "magt5m": <mean annual ground temperature value at 5 m depth (°C)>,
        "magtsurface": <mean annual ground temperature value at 0.01 m depth (°C)>,
        "permafrostbase": <depth of permafrost base (m)>,
        "permafrosttop": <depth of permafrost top (m)>,
        "talikthickness": <thickness of talik layer (m)>
      },
      ...
    },
    ...
  },
  ...
}

Geophysical Institute Permafrost Lab (GIPL) min/mean/max summaries

Results from min/mean/max summary queries will look like this:

{
  "5ModelAvg": {
    "gipl1kmmax": {...},
    "gipl1kmmean": {...},
    "gipl1kmmin": {...},
  },
  ...
}

The above output is structured like this:

{
  <model>: {
    "gipl1kmmax": <max values for all ten variables across all scenarios and years in the time slice>,
    "gipl1kmmean": <mean values for all ten variables across all scenarios and years in the time slice>,
    "gipl1kmmin": <min values for all ten variables across all scenarios and years in the time slice>
  },
  ...
}

Summaries from multiple sources

Results from multiple source summary queries will look like this:

{
  "gipl": {
    "1995": {
      "cruts31": {
        "historical": {
          "alt": 0.59,
          "magt": -2.2
        }
      }
    },
    ...
    "title": "Melvin et al. (2017) GIPL 2.0 Mean Annual Ground Temperature (°C) and Active Layer Thickness (m) Model Output"
  },
  "jorg": {
    "ice": "Moderate",
    "pfx": "Discontinuous",
    "title": "Jorgenson et al. (2008) Permafrost Extent and Ground Ice Volume"
  },
  "obu_magt": {
    "depth": "Top of Permafrost",
    "temp": 0.2,
    "title": "Obu et al. (2018) 2000-2016 Mean Annual Top of Permafrost Ground Temperature (°C)",
    "year": "2000-2016"
  },
  "obupfx": {
    "pfx": "Sporadic",
    "title": "Obu et al. (2018) Permafrost Extent"
  }
}

The above output is structured like this:

{
  "gipl": {
    <era>1: {
      <model>: {
        <scenario>: {
          "alt": <active layer thickness (m)>,
          "magt": <mean annual ground temperature (°C)>
        },
        ...
      },
      ...
    },
    ...
    "title": <title describing these results>
  },
  "jorg": {
    "ice": <estimated ground ice volume>,
    "pfx": <permafrost extent>,
    "title": <title describing these results>
  },
  "obu_magt": {
    "depth": <top of permafrost (i.e. depth of active layer)>,
    "temp": <mean annual ground temperature (°C)>,
    "title": <title describing these results>,
    "year": <date range of model output>
  },
  "obupfx": {
    "pfx": <permafrost extent>,
    "title": <title describing these results>
  }
}
1 Note for GIPL eras: "1995" refers to the historical baseline. Otherwise these are 30 year eras surrounding the following center dates, except the last era is truncated at 2100: "2025" (2011-2040), "2050" (2036-2065), "2075" (2061–2090), "2095" (2086–2100).
