CMIP6, Downscaled

This endpoint provides access to daily CMIP6 data that have been bias-adjusted and downscaled to 4km using dynamically downscaled ERA5 reference data. Twelve models are available with up to four future scenarios available per model. This is a subset of the model and scenario combinations described here. A 6-model average is also available. This dataset spans years 1965–2100 and its extent covers most of Alaska and parts of western Canada.

Service endpoints

CMIP6 downscaled daily data

Query data for provided models, scenarios, and variables. These models, scenarios, and variables are defined below.

Endpoint Example URL
Values for all models, scenarios, and variables at a given latitude and longitude /cmip6_downscaled/point/61.5/-147
Values at a given latitude and longitude for provided model, scenario, and variable /cmip6_downscaled/point/61.5/-147?models=6ModelAvg&scenarios=ssp585&vars=tasmax
Multiple models can be requested by appending a comma-separated string like ?models=<model_1>,<model_2>.
Multiple scenarios can be requested by appending a comma-separated string like ?scenarios=<scenario_1>,<scenario_2>.
Multiple variables can be requested by appending a comma-separated string like ?vars=<var_id_1>,<var_id_2>.
CSV output is also available by appending ?format=csv to the URL.
Chaining of multiple arguments is supported (e.g., ?vars=tasmax,tasmin&format=csv).

CMIP6 Variable Information

Below is the list of CMIP6 variables that are found in this endpoint. For complete variable definitions, see the official CMIP6 documentation here.

pr
Precipitation, including both liquid and solid phases (mm/day)
tasmax
Maximum near surface air temperature (°C)
tasmin
Minimum near surface air temperature (°C)
