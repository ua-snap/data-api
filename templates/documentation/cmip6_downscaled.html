CMIP6, Downscaled

This endpoint provides access to daily CMIP6 data that have been bias-adjusted and downscaled to 4km using dynamically downscaled ERA5 reference data. Select models and scenarios are available. Read more about our CMIP6 subset here. This dataset spans years 1965–2100 and its extent covers most of Alaska and parts of western Canada. A 6-model average is also available, consisting of an equal-weighted mean of the KACE-1-0-G, MIROC6, MPI-ESM1-2-HR, MRI-ESM2-0, NorESM2-MM, and TaiESM1 models.

Service endpoints

CMIP6 downscaled daily data

Query data for provided models, scenarios, and variables. These variables are defined below.

Endpoint Example URL
Daily values at a given latitude and longitude /cmip6_downscaled/point/61.5/-147?models=6ModelAvg&scenarios=ssp585&vars=tasmax
Multiple models can be requested by appending a comma-separated string like ?models=<model_1>,<model_2>.
Multiple scenarios can be requested by appending a comma-separated string like ?scenarios=<scenario_1>,<scenario_2>.
Multiple variables can be requested by appending a comma-separated string like ?vars=<var_id_1>,<var_id_2>.
CSV output is also available by appending ?format=csv to the URL.
Chaining of multiple arguments is supported (e.g., ?vars=tasmax,tasmin&format=csv).

CMIP6 Variable Information

Below is the list of CMIP6 variables that are found in this endpoint. For complete variable definitions, see the official CMIP6 documentation here.

pr
Precipitation, including both liquid and solid phases (kg m-2 s-1)
tasmax
Maximum near surface air temperature (°C)
tasmin
Minimum near surface air temperature (°C)
