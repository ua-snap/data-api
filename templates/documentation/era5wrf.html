WRF Dynamically Downscaled ERA5 Reanalysis

This endpoint provides access to dynamically downscaled (using the Weather Research and Forecasting (WRF) model) ERA5 reanalysis climate data. The dataset spans 1960-2023 with daily temporal resolution and 4 km spatial resolution. Available data include temperature, humidity, precipitation, wind, and sea ice variables. To produce this dataset, ERA5 was used to initialize the WRF model version 4.3.3 over the central Alaska landmass and immediately neighboring Canada. Data were produced in 2-day runs, each with a 6-hour spin-up phase, and spectral nudging enabled.

Service endpoints

Point query

Endpoint Example URL
ERA5-WRF point query (all variables, all dates) /era5wrf/point/65.0628/-146.1627
ERA5-WRF point query (single variable, all dates) /era5wrf/point/65.0628/-146.1627?vars=t2_mean
ERA5-WRF point query (several variables, all dates) /era5wrf/point/65.0628/-146.1627?vars=t2_mean,rh2_mean
CSV output is available by appending ?format=csv to the URL.
Specific variables can be requested using ?vars=t2_mean,rh2_mean.
Chaining parameters is also supported, e.g. ?vars=t2_mean,rh2_mean&format=csv.
All parameter combinations work for both point and area queries.

Point Query Output

Data are returned as JSON by default, keyed by date, with each date containing a dictionary of variable names and values. The variable names are the same as the ones listed in the Available Variables section.

  },

    "rh2_mean": 79.3,

    "rh2_mean": 63.5,

}

Area query

Area queries return zonal statistics (mean values) aggregated over a specified polygon area. Values represent the spatial average of all 4 km by 4 km grid cells within the polygon boundary. Learn more about what types of places are available for area queries.

Endpoint Example URL
Return mean values over area for all variables, full time range /era5wrf/area/19070506
Return mean values over area for selected variables /era5wrf/area/19070506?vars=t2_mean,rh2_mean
Return mean values over area for selected variables in CSV format /era5wrf/area/19070506?vars=t2_min,t2_mean,t2_max&format=csv
Return mean values over area for selected variables /era5wrf/area/19070506?vars=t2_mean,rh2_mean
Area queries support the same parameters as point queries: vars (comma-separated variable names), and format=csv for CSV output.
Results represent mean values of all 4 km by 4 km grid cells within the specified polygon area.

Area Query Output

Area queries return the same JSON structure as point queries, but values represent spatial averages rather than point measurements: 

{
  "2012-06-01": {
    "rh2_mean": 85.7,
    "t2_mean": 8.2
  },
  "2012-06-02": {
    "rh2_mean": 82.1,
    "t2_mean": 9.5
  },
  "2012-06-03": {
    "rh2_mean": 78.9,
    "t2_mean": 10.1
  },
  ...
}

Each value is the mean of all valid grid cells within the polygon boundary for that date.
