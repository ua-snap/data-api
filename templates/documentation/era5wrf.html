WRF Dynamically Downscaled ERA5 Reanalysis

This endpoint provides access to dynamically downscaled (using the Weather Research and Forecasting (WRF) model) ERA5 reanalysis climate data. The dataset spans 1960-2019 with daily temporal resolution and 4 km spatial resolution. Available data include temperature, humidity, precipitation, wind, and sea ice variables. To produce this dataset, ERA5 was used to initialize the WRF model version 4.3.3 over the central Alaska landmass and immediately neighboring Canada. Data were produced in 2-day runs, each with a 6-hour spin-up phase, and spectral nudging enabled.

Service endpoints

Point query

Endpoint Example URL
ERA5-WRF point query (all variables, all dates) /era5wrf/point/65.0628/-146.1627
ERA5-WRF point query (single variable, all dates) /era5wrf/point/65.0628/-146.1627?vars=t2_mean
ERA5-WRF point query (several variables, one date) /era5wrf/point/65.0628/-146.1627?vars=t2_mean,rh2_mean&start_date=2012-06-01&end_date=2012-06-01
ERA5-WRF point query (single variable, date range) /era5wrf/point/65.0628/-146.1627?vars=t2_mean&start_date=2012-06-01&end_date=2012-06-30
Date ranges can be specified using start_date=YYYY-MM-DD and end_date=YYYY-MM-DD query parameters.
CSV output is available by appending ?format=csv to the URL.
Specific variables can be requested using ?vars=t2_mean,rh2_mean.
Chaining parameters is also supported, e.g. ?vars=t2_mean,rh2_mean&format=csv.

Point Query Output

Data are returned as JSON by default, keyed by date, with each date containing a dictionary of variable names and values. The variable names are the same as the ones listed in the Available Variables section.
