Pan-Arctic Sea Ice Concentration

Query pan-arctic sea ice concentration information for the Arctic.

Service endpoints

Point query

This endpoint accesses historical monthly data on sea ice concentration in the Arctic from 1850–2021.

Endpoint Example URL
Sea ice concentration point query /seaice/point/63.73/-166.32
CSV output is also available by appending ?format=csv to the URL.

Output

Results from sea ice concentration queries will look like this:

{
  "1850-01": 100,
  "1850-02": 98,
  "1850-03": 96,
  ...
}

The above output is structured like this:

{
  "<year>-<month>": <sea ice concentration for one of the 12 months in a year>,
  ...
}
