Pan-Arctic Sea Ice Concentration

This endpoint provides access to monthly pan-Arctic sea ice concentration data at a resolution of approximately 17.1km for years 1850–2021. Data were provided by Walsh et al.

Links to an academic reference and source dataset are included at the bottom of this page.

Service endpoints

Most recent month and year of available data

Query the metadata to get the most recent month and year of the available data.

Endpoint Example URL
Most recent month and year of available data /seaice/enddate

Point query

Query historical monthly pan-Arctic sea ice concentration data from 1850–2021 at a single point specified by latitude and longitude.

Endpoint Example URL
Sea ice concentration point query /seaice/point/63.73/-166.32
CSV output is also available by appending ?format=csv to the URL.

Output

Results from sea ice concentration queries will look like this:

{
  "1850-01": 100,
  "1850-02": 98,
  "1850-03": 96,
  ...
}

The above output is structured like this:

{
  "<year>-<month>": <sea ice concentration for one of the 12 months in a year>,
  ...
}
