Pan-Arctic Sea Ice Concentration

This endpoint provides access to pan-Arctic sea ice concentration data.

Service endpoints

Point query

Query historical monthly pan-Arctic sea ice concentration data from 1850–2021 at a single point specified by latitude and longitude.

Endpoint Example URL
Sea ice concentration point query /seaice/point/63.73/-166.32
CSV output is also available by appending ?format=csv to the URL.

Output

Results from sea ice concentration queries will look like this:

{
  "1850-01": 100,
  "1850-02": 98,
  "1850-03": 96,
  ...
}

The above output is structured like this:

{
  "<year>-<month>": <sea ice concentration for one of the 12 months in a year>,
  ...
}

Source data

Walsh, J.E., Fetterer, F., Scott Stewart, J. and Chapman, W.L., 2017. A database for depicting Arctic sea ice variations back to 1850. Geographical Review, 107(1), pp.89-107, doi:10.1111/j.1931-0846.2016.12195.x.
