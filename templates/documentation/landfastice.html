Landfast Sea Ice Extent

This endpoint accesses daily landfast ice extent information for the Alaska coastline between October 17th, 1996 and July 14th, 2008.

Service endpoints

Point query

Query the daily landfast ice extent for a certain geographic point:

Endpoint Example URL
Landfast sea ice extent point query /landfastice/point/70.6123/-149.8869
CSV output is also available by appending ?format=csv to the URL.

Output

Results from point queries will look like this:

{
  "01-01-1997": null,
  "01-01-1998": null,
  "01-01-1999": null,
  "01-01-2000": 1,
  "01-01-2001": 1,
  "01-01-2002": 1,
  "01-01-2003": 1,
  "01-01-2004": 1,
  "01-01-2005": 1,
  "01-01-2006": 1,
  "01-01-2007": 1,
  "01-01-2008": null,
  ...
}

The above output is structured like this:

{
  "<month>-<day>-<year>": <landfast ice presence (1) or absence (null) for a specific date in the time series>,
  ...
}

Source data

These data are part of the Arctic Data Collaborative (ARDAC). Scientists working on Arctic research need easily obtainable climate and environmental data. ARDAC is a system of online technologies that curate and distribute relevant Arctic environmental data in formats that improve the ability of scientists to use this data. Users of these data should cite:

Mahoney, A., Eicken, H., Gaylord, A. G., and Shapiro, L. (2007), Alaska landfast sea ice: Links with bathymetry and atmospheric circulation, J. Geophys. Res., 112, C02001, doi:10.1029/2006JC003559.
