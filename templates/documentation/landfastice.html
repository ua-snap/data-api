Landfast Sea Ice Extent

This endpoint provides access to daily landfast sea ice extent data at a spatial resolution of 100 m. Data are available October 1996 – July 2023 for the Beaufort Sea region and the Chukchi Sea region.

Links to academic references are included at the bottom of this page.

Service endpoints

Point query

Query the daily landfast sea ice extent for a single point specified by latitude and longitude.

Endpoint Example URL
Landfast sea ice extent point query /landfastice/point/70.6123/-149.8869
CSV output is also available by appending ?format=csv to the URL.

Output

Results from point queries will look like this:

{
  "1996-10-27": 255,
  "1996-10-28": 255,
  "1996-10-29": 255,
  ...
  "2023-07-29": 0,
  "2023-07-30": 0,
  "2023-07-31": 0,
}

The above output is structured like this:

{
  "<year>-<month>-<day>": <landfast ice presence (255) or absence (0) or grid cell classified as land (128) for a specific date in the time series>,
  ...
}
