Species

The endpoints here provide access to modeled species distribution data.

These data were modeled using Hydrologic Unit (Level 12), or HUC-12, polygons as the spatial unit of analysis. Querying a point will return data for the closest HUC-12 polygon used in the dataset.

See the LINK TO GEONETWORK PAGE page for more information. Links to academic references are included at the bottom of this page.

Service endpoints

Intersecting area query

Query data derived for a point location specified by latitude and longitude, and return the modeled species distribution data from the HUC-12 polygon that intersects the supplied point location.

Endpoint Example URL
Species data intersecting area query /species/65.24/-142.22
CSV output is also available by appending ?format=csv to the URL.

Output

Intersecting area query

Results from intersecting area queries will look like this:

    {
        "amphibians": [
          [
            "Wood Frog",
            "Lithobates sylvaticus"
          ],
          ...
        ],
        "birds": [
          [
            "Alder Flycatcher",
            "Empidonax alnorum"
          ],
          ...
          [
            "Yellow Warbler",
            "Setophaga petechia"
          ]
        ],
        "mammals": [
          [
            "Arctic Ground Squirrel",
            "Urocitellus parryii"
          ],
          ...
          [
            "Tundra Shrew",
            "Sorex tundrensis"
          ]
        ],
        "total_sgcn_species": "92"
      }

The above output is structured like this:

{
    "amphibians" : <list of tuples containing common and scientific names of amphibian species>,
    "birds" : <list of tuples containing common and scientific names of bird species>,
    "mammals" : <list of tuples containing common and scientific names of mammal species>,
    "total_sgcn_species" : <count of all SGCN species>,
    }
