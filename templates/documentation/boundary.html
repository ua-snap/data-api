Physiographic and Administrative Boundary Data

These endpoints provide access to a common set of areas, and their respective polygons, used across SNAP tools and data.

Service endpoints

Available areas

Query to see a list of available areas by type.

Endpoint Example URL
All available areas of interest /areas/all
Hydrological unit codes (HUCs) /areas/hucs
Alaska, British Columbia, or Yukon protected areas /areas/protected_areas
Alaska Native corporations /areas/corporations
Alaska climate divisions /areas/climate_divisions
Fire management zones /areas/fire_zones
Alaska ethnolinguistic regions /areas/ethnolinguistic_regions
Alaska boroughs (county equivalent) /areas/boroughs
Alaska census areas for the Unorganized Borough /areas/census_areas
Game management units /areas/game_management_units
First nations /areas/first_nations
CSV output is also available by appending ?format=csv to the URL.

GeoJSON polygons

Query for GeoJSON polygon(s) by adding geojson/<id> to the base URL, using an identifier from one of the lists above.

For example: /geojson/19070506
