CONUS Hydrology

These endpoints provide access to modeled streamflow statistics and daily streamflow climatologies for 56,460 stream network segments in the continental United States. Stream segments were derived from the National Hydrologic Model Geospatial Fabric, and streamflow was modeled using the Precipitation-Runoff Modeling System (PRMS) and CMIP5 historical and projected climate data. A calibrated historical baseline (the Maurer calibration) is available for the 1976–2005 period.

For more detailed model information, see the links to academic references and source datasets included at the bottom of this page.

We also include endpoints to access daily streamflow climatologies from associated USGS stream gauge data collected over the 1976–2005 historical period, for use in comparing with the modeled data. Note that not all modeled stream segments have associated gauge data. See the /conus_hydrology/gauge_info endpoint for a list of all stream IDs with gauge data.

Service endpoints

Modeled hydrologic statistics by stream ID

Query hydrologic statistics for all models and scenarios. Statistics are computed over historical (1976–2005) and future (2016–2045, 2046–2075, and 2071–2100) eras.

Endpoint Example URL
Hydrologic statistics /conus_hydrology/stats/50563
CSV output is also available by appending ?format=csv to the URL.

Modeled daily streamflow climatologies by stream ID

Query daily streamflow climatologies for all models and scenarios. Useful for constructing hydrographs. Minimum, maximum, and mean values for each day of year are computed over historical (1976–2005) and future (2016–2045, 2046–2075, and 2071–2100) eras.

Endpoint Example URL
Modeled daily streamflow climatologies /conus_hydrology/modeled_climatology/50563
CSV output is also available by appending ?format=csv to the URL.

Observed daily streamflow climatologies by stream ID

Query daily streamflow climatologies computed from observed USGS stream gauge data. Useful for constructing hydrographs. Climatologies are computed over the 1976–2005 historical era.

Endpoint Example URL
Observed daily streamflow climatology /conus_hydrology/observed_climatology/50563
CSV output is also available by appending ?format=csv to the URL.
See /conus_hydrology/gauge_info for a list of all stream IDs with gauge data.
