CONUS Hydrology

These endpoints provide access to modeled streamflow statistics and daily streamflow climatologies for 56,460 stream network segments in the continental United States. Stream segments were derived from the National Hydrologic Model Geospatial Fabric, and streamflow was modeled using the Precipitation-Runoff Modeling System (PRMS) and CMIP5 historical and projected climate data. A calibrated historical baseline (the Maurer calibration) is available for the 1976-2005 period.

For more detailed model information, see the links to academic references and source datasets included at the bottom of this page.

We also include endpoints to access daily streamflow climatologies from associated USGS stream gauge data collected over the 1976-2005 historical period, for use in comparing with the modeled data. Note that not all modeled stream segments have associated gauge data. See the /conus_hydrology/gauge/info endpoint for a list of all stream IDs with gauge data.
