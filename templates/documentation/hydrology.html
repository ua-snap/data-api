Hydrology

These endpoints provide access to historical and projected modeled decadal summaries of hydrologic data for Alaska at a resolution of XXXm. Historical data were derived from the XXX dataset. Projections were derived from XXX, YYY, and ZZZ model outputs under the RCP 4.5 and RCP 8.5 emissions scenarios.

Links to academic references and source datasets are included at the bottom of this page.

Service endpoints

All hydrologic variables (point query)

Query all hydrologic variables for all models and scenarios for each decade.

Endpoint Example URL
Decadal variables /hydrology/point/61.5/-147
CSV output is available by appending ?format=csv to the URL.

Output

Decadal hydrology variables

  {
    "ACCESS1-3": {
      "rcp45": {
        "apr": {
          "1950-1959": {
            "evap": "3",
            "glacier_melt": "0",

The above output is structured like this:

{
  <model>: {
    <scenario>: {
      <month>: {
        <decade>: {
          "evap": <monthly sum of daily evapotranspiration, mm>,
          "glacier_melt": <monthly sum of daily glacier ice melt, mm>,
          "iwe": <monthly maximum of daily ice water equivalent, mm>,
          "pcp": <monthly sum of daily precipitation, mm>,
          "runoff": <monthly sum of daily surface runoff, mm>,
          "sm1": <monthly mean of daily soil moisture in layer 1, mm>,
          "sm2": <monthly mean of daily soil moisture in layer 2, mm>,
          "sm3": <monthly mean of daily soil moisture in layer 3, mm>,
          "snow_melt": <monthly sum of daily snowmelt, mm>,
          "swe": <monthly maximum of daily snow water equivalent, mm>,
          "tmax": <monthly mean of daily maximum air temperature at 2m, degrees C>,
          "tmin": <monthly mean of daily minimum air temperature at 2m, degrees C>,
        },
        ...
      },
      ...
    },
    ...
  },
  ...
}
