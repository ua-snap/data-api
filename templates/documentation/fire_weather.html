CMIP6 Fire Weather Indices

This endpoint provides access to fire weather indices derived from bias-corrected CMIP6 daily data (for historical and future periods) and from the ERA5 reanalysis (for the historical period). The dataset spans 1980–2100 with daily temporal resolution from April 1 to October 31 of each year, at 0.25 x 0.25 degree spatial resolution. The spatial domain is limited to the North American boreal ecoregion, as defined by Olson et al. (2001). Variables are those used in the Canadian Forest Fire Danger Rating System (CFFDRS) Canadian Forest Fire Weather Index (FWI) System. Read more about CFFDRS and FWI here.

Available Variables

Variable id Description
Buildup Index bui Numeric rating of the total amount of fuel available for combustion. It is based on the dmc and the dc.
Drought Code dc Numeric rating of the average moisture content of deep, compact organic layers. This code is a useful indicator of seasonal drought effects on forest fuels and the amount of smoldering in deep duff layers and large logs.
Duff Moisture Code dmc Numeric rating of the average moisture content of loosely compacted organic layers of moderate depth. This code gives an indication of fuel consumption in moderate duff layers and medium-size woody material.
Fine Fuel Moisture Code ffmc Numeric rating of the moisture content of litter and other cured fine fuels. This code is an indicator of the relative ease of ignition and the flammability of fine fuel.
Fire Weather Index fwi Numeric rating of fire intensity. It is based on the isi and the bui, and is used as a general index of fire danger throughout the forested areas of Canada and Alaska.
Initial Spread Index isi Numeric rating of the expected rate of fire spread. It is based on wind speed and ffmc. Like the rest of the FWI system components, isi does not take fuel type into account. Actual spread rates vary between fuel types at the same isi.

Available Operations

Operation id Description
Compute n-day Rolling Average 3_day_rolling_average, 5_day_rolling_average, 7_day_rolling_average Computes a rolling average of daily values using an n-day window centered on each day of the year. The maximum, mean, and minimum of rolling average values across the selected year range are returned. For area queries, rolling average values are weighted by spatial area.
Count Summer Fire Danger Rating Days summer_fire_danger_rating_days The mean number of summer (Juneâ€“August) days per year, over the selected year range, classified into fire danger categories (low, moderate, high, very high, extreme) according to the fire behavior index thresholds. (See Table 1 of the Alaska Interagency Fire Danger Operating Plan for threshold values.) For area queries, daily classifications are based on the area-weighted mean for each variable.

Service endpoints

Point query

Endpoint Example URL
CMIP6 Fire Weather Indices point query (select years, all variables, 3 day rolling average operation) /fire_weather/point/64.8/-147.5/1980/2010?op=3_day_rolling_average
CMIP6 Fire Weather Indices point query (select years, single variable, summer fire danger rating days operation) /fire_weather/point/64.8/-147.5/2030/2050?vars=bui&op=summer_fire_danger_rating_days
CMIP6 Fire Weather Indices point query (select years, several variables, 7 day rolling average operation) /fire_weather/point/64.8/-147.5/2030/2050?vars=bui,fwi,isi&op=7_day_rolling_average
Only one operation is allowed per request. If no operation is requested, default is 3_day_rolling_average.
CSV output is available by appending ?format=csv to the URL.
Multiple variables can be requested using ?vars=dc,dmc,ffmc.
Chaining parameters is also supported, e.g. ?vars=t2_mean,rh2_mean&format=csv.

Point Query Output

n-day Rolling Averages

Data are returned as JSON keyed hierarchically by selected year range, variable, model, day of year, and statistic. Results are the maximum, mean, and minimum of the n-day rolling average values across the selected year range.

Area query

Area queries return zonal statistics aggregated over a specified polygon area. Values represent the spatially-weighted average of all 0.25 x 0.25 degree grid cells within the polygon boundary. Learn more about what types of places are available for area queries.

Endpoint Example URL
CMIP6 Fire Weather Indices area query (select years, all variables, 3 day rolling average operation) /fire_weather/area/19080309/1980/2010?op=3_day_rolling_average
