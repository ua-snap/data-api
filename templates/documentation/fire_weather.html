CMIP6 Fire Weather Indices

This endpoint provides access to fire weather indicies derived from bias-corrected CMIP6 daily data (for historical and future periods) and from the ERA5 reanalysis (for the historical period). The dataset spans 1980–2100 with daily temporal resolution and 0.25 x 0.25 degree spatial resolution. The spatial domain is limited to the North American boreal ecoregion, as defined by Olson et al. (2001). Variables are those used in the Canadian Forest Fire Danger Rating System (CFFDRS) Canadian Forest Fire Weather Index (FWI) System.

Available Variables

Variable id Description
Build-Up Index bui Numeric rating of the total amount of fuel available for combustion. It is based on the dmc and the dc.
Drought Code dc Numeric rating of the average moisture content of deep, compact organic layers. This code is a useful indicator of seasonal drought effects on forest fuels and the amount of smoldering in deep duff layers and large logs.
Duff Moisture Code dmc Numeric rating of the average moisture content of loosely compacted organic layers of moderate depth. This code gives an indication of fuel consumption in moderate duff layers and medium-size woody material.
Fine Fuel Moisture Code ffmc Numeric rating of the moisture content of litter and other cured fine fuels. This code is an indicator of the relative ease of ignition and the flammability of fine fuel.
Fire Weather Index fwi Numeric rating of fire intensity. It is based on the isi and the bui, and is used as a general index of fire danger throughout the forested areas of Canada and Alaska.
Initial Spread Index isi Numeric rating of the expected rate of fire spread. It is based on wind speed and ffmc. Like the rest of the FWI system components, isi does not take fuel type into account. Actual spread rates vary between fuel types at the same isi.

Service endpoints

Point query

Endpoint Example URL
CMIP6 Fire Weather Indices point query (3 day rolling average, all variables, all years) /fire_weather/point/64.8/-147.5?op=3_day_rolling_average
CMIP6 Fire Weather Indices point query (summer fire danger rating days, single variable, all years) /fire_weather/point/65.0628/-146.1627?op=summer_fire_danger_rating_days&vars=bui
CMIP6 Fire Weather Indices point query (7 day rolling average, several variables, select years) /fire_weather/point//64.8/-147.5/2030/2050?op=7_day_rolling_average&vars=bui,fwi,isi
Only one operation is allowed per request.
CSV output is available by appending ?format=csv to the URL.
Multiple variables can be requested using ?vars=dc,dmc,ffmc.
Chaining parameters is also supported, e.g. ?vars=t2_mean,rh2_mean&format=csv.
All parameter combinations work for both point and area queries.
