TTY

These endpoints provide access to TTY data derived from historical reanalysis.

Links to academic references and a source dataset are included at the bottom of this page.

Service endpoints

Point query

Query TTY.

Output

Results from TTY queries will look like this:

{
  "historical": {
    "wdpymax": 160,
    "wdpymean": 134,
    "wdpymin": 112
  },
  ...
}

The above output is structured like this:

{
  <era>: {
    "wdpymax": <max number of wet days per year across all models and years used for the summary era time period>,
    "wdpymean": <number of wet days per year across all models and years used for the summary era time period>,
    "wdpymin": <number of wet days per year across all models and years used for the summary era time period>
  },
  ...
}
