Glacier Data

These data are for use in the Arctic-EDS.

Service endpoints

Point query

Query the glaciology of Alaska for a single point specified by latitude and longitude.

Endpoint Example URL
Glacier point query /glacier/point/60.606/-143.345

Output

Results from point queries will look like this:

{
  "ewisc": true,
  "lwisc": true,
  "now": {
    "glacname": "Steller Glacier",
    "modern": true
  },
  "pmax": true,
  "title": "Past and Present Glaciology of Alaska"
}

The above output is structured like this:

{
  "ewisc": <the Early Wisconsin Glaciation did/did not cover this point with ice>,
  "lwisc": <the Late Wisconsin Glaciation did/did not cover this point with ice>,
  "now": {
    "glacname": <name of glacier or ice field, if available>,
    "modern": <the contemporary glacial extent does/does not cover this point with ice>
  },
  "pmax": <the Pleistocene Maximum Glacial Extent did/did not cover this point with ice>,
  "title": <title describing these results>
}
