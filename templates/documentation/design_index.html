Design Index

These endpoints provide access to historical and projected design thawing index and design freezing index values at a resolution of approximately 18.5km. A design index is the mean of the three highest cumulative annual degree days for a given temperature threshold. Historical data are derived from the ERA-Interim dataset and are summarized across years 1980–2009. Projections are derived from GFDL-CM3 and NCAR-CCSM4 model outputs using the RCP 8.5 emissions scenario and are summarized across years 2040–2069 and 2070–2099.

A link to the source dataset is included at the bottom of this page.

Datasets

Dataset Description
Thawing The mean of the three highest cumulative annual degree days from the thawing index for each era.
Freezing The mean of the three highest cumulative annual degree days from the freezing index for each era.

Output

Summary across all models

Point query summaries across all models will look like this:

{
  "2040-2069": {
    "di": 3787
  },
  ...
}

The above output is structured like this:

{
  <era>: {
    "di": <mean of degree days across top three years across models and era>
  },
  ...
}

Results from each model

Point query results from each model will look like this:

{
  ...
  "GFDL-CM3": {
    "2040-2069": {
      "di": 1548
    },
    "2070-2099": {
      "di": 775
    }
  },
  ...
}

The above output is structured like this:

{
  <model>: {
    <era>: {
      "di": <mean of degree days across top three years across era>
    },
    ...
  },
  ...
}
