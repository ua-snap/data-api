Temperature Anomalies

This endpoint provides access to historical and projected annual mean temperature anomalies, which are the differences in annual mean temperature compared to a 1951–1980 baseline. Historical temperature anomalies are provided by Berkeley Earth's global monthly land and ocean dataset at a resolution of 0.25째 longitude x 0.25째 latitude (currently in beta). Projected temperature anomalies are provided by multiple CMIP6 models for multiple scenarios at a resolution of 100 km.

Service endpoints

Point query

Query annual mean temperature anomalies for each year from 1850–2100, and their corresponding temperature baselines, at a single point specified by latitude and longitude.

Endpoint Example URL
Temperature anomalies point query /temperature_anomalies/point/63.73/-166.32
CSV output is also available by appending ?format=csv to the URL.

Output

Results from temperature anomalies queries will look like this:

{
  "Berkeley-Earth": {
    "temperature_anomalies": {
      "historical": {
        "1850": -0.73,
        "1851": 0.13,
        "1852": -0.19,
        ...
      },
    },
    "temperature_baseline": -6.04
  },
  "CESM2": {
    "temperature_anomalies": {
      "ssp126": {
        "2025": 3.44,
        "2026": 1.51,
        "2027": 2.24,
        ...
      },
      ...
    },
    "temperature_baseline": -4.18
  },
  ...
}

The above output is structured like this:

{
  <model>: {
    "temperature_anomalies": {
      <scenario>: {
        <year>: <difference from model's temperature_baseline in 째C>,
        ...
      },
    },
    "temperature_baseline": <model's temperature mean for 1951-1980 in 째C>
  },
}
