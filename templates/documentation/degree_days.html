Degree Days Min, Mean, and Max Point Query

These endpoints provide access to historical and projected total annual cumulative degree days for several different degree day thresholds.

Service endpoints

Min/mean/max summaries (point query)

Query the minimum, mean, and maximum degree days values summarized across all models and years.

Endpoint Example URL
Historical degree days below 65°F min/mean/max /mmm/degree_days/heating/historical/65.0628/-146.1627
Projected degree days below 65°F min/mean/max /mmm/degree_days/heating/projected/65.0628/-146.1627
Historical and projected degree days below 65°F min/mean/max /mmm/degree_days/heating/hp/65.0628/-146.1627
Historical degree days below 0°F min/mean/max /mmm/degree_days/below_zero/historical/65.0628/-146.1627
Projected degree days below 0°F min/mean/max /mmm/degree_days/below_zero/projected/65.0628/-146.1627
Historical and projected degree days below 0°F min/mean/max /mmm/degree_days/below_zero/hp/65.0628/-146.1627
Historical degree days above 32°F min/mean/max /mmm/degree_days/thawing_index/historical/65.0628/-146.1627
Projected degree days above 32°F min/mean/max /mmm/degree_days/thawing_index/projected/65.0628/-146.1627
Historical and projected degree days above 32°F min/mean/max /mmm/degree_days/thawing_index/hp/65.0628/-146.1627

Min/mean/max summaries across provided year range (point query)

Query the minimum, mean, and maximum degree days values summarized across all models and years for the specified year range. Valid year ranges are between 1980–2009 for historical data and 2006–2100 for projected data.

Degree days for all years (point query)

Query degree days values for each model and year.

Endpoint Example URL
Historical and projected degree days below 65°F /mmm/degree_days/heating/all/65.0628/-146.1627
Historical and projected degree days below 0°F /mmm/degree_days/below_zero/all/65.0628/-146.1627
Historical and projected degree days above 32°F /mmm/degree_days/thawing_index/all/65.0628/-146.1627
Historical and projected degree days below 32°F /mmm/degree_days/freezing_index/all/65.0628/-146.1627
CSV output is also available by appending ?format=csv to the URL.

Output

Min/mean/max summaries (point query)

Results from min/mean/max queries will look like this:

{
  "historical": {
    "ddmax": 14972,
    "ddmean": 13696,
    "ddmin": 9357
  },
  ...
}

The above output is structured like this:

{
  <"historical" or "projected">: {
    "ddmax": <max dd value across all models and years>,
    "ddmean": <mean dd value across all models and years>,
    "ddmin": <min dd value across all models and years>
  },
  ...
}

Degree days for all years (point query)

Results from yearly degree days queries will look like this:

{
  "ERA-Interim": {
    "1980": {
      "dd:" 4802,
    },
    ...
  },
  ...
}

The above output is structured like this:

{
  <model>: {
    <year>: {
      "dd": <degree days value>,
    },
    ...
  },
  ...
}
