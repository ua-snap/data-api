Snowfall Equivalent

This endpoint accesses historical and projected modeled decadal summaries of annual snowfall equivalent (SFE) for Alaska. Note that SFE has a very specific meaning in a modeling context: it is the product of the fraction of days in a month with precipitation falling as snow and the total monthly precipitation. That product (monthly) is then summed to create a single annual total SFE value. The SFE values provided here are millimeters of water.

Service endpoints

Min/mean/max summaries (point query)

The following queries will return SFE minimum, mean, and maximum across all models, scenarios, and decades.

Endpoint Example URL
Historical SFE min/mean/max /mmm/snow/snowfallequivalent/historical/61.5/-147
Projected SFE min/mean/max /mmm/snow/snowfallequivalent/projected/61.5/-147
Historical and projected SFE min/mean/max /mmm/snow/snowfallequivalent/hp/61.5/-147

Snowfall equivalent for all decades (point query)

The following query will return SFE for all models, scenarios, and decades (1910-1919 through 2090-2099):

Endpoint Example URL
SFE for all decades /mmm/snow/snowfallequivalent/all/61.5/-147
CSV output is also available by appending ?format=csv to the URL.

Output

Min/mean/max summaries

Results from SFE min/mean/max queries will look like this:

{
  "historical": {
    "sfemax": 4876,
    "sfemean": 3701,
    "sfemin": 2946
  },
  ...
}

The above output is structured like this:

{
  <"historical" or "projected">: {
    "sfemax": <max SFE value across all models, scenarios, and decades>,
    "sfemean": <mean SFE value across all models, scenarios, and decades>,
    "sfemin": <min SFE value across all models, scenarios, and decades>
  },
  ...
}

Snowfall equivalent for all decades

Results from SFE for all decades queries will look like this:

{
  "CRU-TS": {
    "historical": {
      "1910-1919": {
        "SFE": 3801
      },
      ...
    },
    ...
  },
  ...
}

The above output is structured like this:

{
  <model>: {
    <scenario>:{
      <decade>: {
        "SFE": <annual total sfe value>
      },
      ...
    },
    ...
  },
  ...
}
