Mean Annual Precipitation

Query mean annual precipitation projections for Alaska.

Service endpoints

Point query

Query mean annual precipitation projections for a single point specified by latitude and longitude.

Endpoint Example URL
Mean annual precipitation point query /mean_annual_precip/point/65.0628/-146.1627

Output

Results from point queries will look like this:

{
  "dec_mapr": [
    {
      "dec_end": "2059",
      "dec_start": "2050",
      "model": "5modelAvg",
      "pr_mm": 408,
      "scenario": "rcp60",
      "title": "Mean Annual Precipitation (mm) Decadal Summary Projection for 2050-2059"
    },
    {
      "dec_end": "2059",
      "dec_start": "2050",
      "model": "5modelAvg",
      "pr_mm": 436,
      "scenario": "rcp85",
      "title": "Mean Annual Precipitation (mm) Decadal Summary Projection for 2050-2059"
    }
  ]
}

The above output is structured like this:

{
  "dec_mapr": [
    {
      "dec_end": <end of decade calendar year>,
      "dec_start": <start of decade calendar year>,
      "model": <climate model used to generate outputs>,
      "pr_mm": <mean annual precipitation (mm)>,
      "scenario": <RCP used to generate outputs>,
      "title": <title describing these results>
    },
    ...
  ]
}
