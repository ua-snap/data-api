Ecoregions Data

This endpoint provides access to EPA Level III Ecoregions of Alaska.

Links to an academic reference and source dataset are included at the bottom of this page.

Service endpoints

Point query

Query the EPA Level III Ecoregions of Alaska at a single point specified by latitude and longitude.

Endpoint Example URL
Ecoregions point query /ecoregions/point/65.0628/-146.1627

Output

Results from ecoregions queries will look like this:

{
  "name": "Interior Highlands",
  "title": "EPA Level III Ecoregions of Alaska"
}

The above output is structured like this:

{
  "name": <title describing these results>,
  "title": <EPA level III ecoregion name>
}
