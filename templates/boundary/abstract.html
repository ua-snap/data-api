Boundary data

These data are intended for exposing the polygon boundaries used across SNAP tools and data. Each query for a polygon will return the representative GeoJSON.

Service Endpoint

The endpoint is boundary/area/<var_id> for all available area of interest ID codes.

HUC 8 Polygon Query

Query HUC-8 of Alaska and return a GeoJSON string of the polygon.

Example HUC-8 query: /boundary/area/19070506

Protected Area Polygons Query

Query the protected area polygons of Alaska, British Columbia, and Yukon and return a GeoJSON string of the polygon.

Example: /boundary/area/NPS12

Alaska Native Corporation Polygon Query

Query Alaska Native corporation regions and return a GeoJSON string of the polygon.

Example: /boundary/area/NC8

Climate Division Polygon Query

Query the climate divisions of Alaska and return a GeoJSON string of the polygon.

Example: /boundary/area/CD3

Fire Management Polygon Query

Query the fire management zones of Alaska and return a GeoJSON string of the polygon.

Example: /boundary/area/FIRE2

Ethnolinguistic Region Polygon Query

Query ethnolinguistic regions of Alaska and parts of Canada and return a GeoJSON string of the polygon.

Example: /boundary/area/EL4
