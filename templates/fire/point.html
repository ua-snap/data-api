Wildfire Point Query

Example: /fire/point/65.0628/-146.1627

Results from the above query will look like this:

{
  "cfd": {
    "code": 1,
    "color": "#2b83ba",
    "title": "Today's Fire Danger",
    "type": "Low"
  },
  "hist_fire": {
    "Angel Creek - Chena Dome": "1950",
    "WEST FORK CHENA": "2002"
  },
  "is_snow": {
    "is_snow": false,
    "title": "Today's Snow Cover"
  },
  "lc": {
    "code": 8,
    "color": "#b38a33",
    "title": "Land cover types",
    "type": "Temperate or sub-polar shrubland"
  },
  "prf": {
    "flamm": 0.0057,
    "title": "Projected relative flammability"
  }
}
  • cfd: Current Fire Danger
    • code: (int), Numeric fire danger rating, range 1 through 5.
    • color: (str), Hex color code to visualize fire danger.
    • title: (str), Title describing these data.
    • type: (str), Fire danger rating.
  • hist_fire: Fire History in format: "'Fire Name': 'Fire Year'"
  • is_snow: Current Day Snow Cover Status
    • is_snow: (bool), If true, point is covered by snow. Else, false.
    • title: (str), Title describing these data.
  • lc: Land cover types
    • code: (int), Numeric land cover code.
    • color: (str), Hex color code to visualize land cover.
    • title: (str), Title describing these data.
    • type: (str), Land cover description.
  • prf: Projected relative flammability
    • title: (str), Title describing these data.
    • flamm: (float), Numeric flammability value

Source data

See the Alaska Wildfire Explorer tool for more documentation on these data.
