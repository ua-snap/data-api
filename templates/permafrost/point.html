Geophysical Institute Permafrost Lab (GIPL) model output

Query the entire time series of GIPL model outputs (1 km spatial resolution) for ten variables for a single point specified by latitude and longitude.

Example: /permafrost/point/gipl/63.0628/-146.1627

Query a specific time slice of GIPL model outputs (1 km spatial resolution) for ten variables for a single point specified by latitude and longitude.

Example: /permafrost/point/gipl/63.0628/-146.1627/2040/2050

Results from the above queries will look like this:

{
  "5ModelAvg": {
    "2021": {
      "RCP 4.5": {
        "magt0.5m": -2.6,
        "magt1m": -3.1,
        "magt2m": -3.3,
        "magt3m": -3.4,
        "magt4m": -3.5,
        "magt5m": -3.5,
        "magtsurface": -0.0,
        "permafrostbase": 153.2,
        "permafrosttop": 0.7,
        "talikthickness": 0
  ...
}
The above output is structured as such: 
{
  <model>: {
    <year>:{
      <scenario>: {
        "magt0.5m": <mean annual ground temperature value at 0.5 m depth (°C)>,
        "magt1m": <mean annual ground temperature value at 1 m depth (°C)>,
        "magt2m": <mean annual ground temperature value at 2 m depth (°C)>,
        "magt3m": <mean annual ground temperature value at 3 m depth (°C)>,
        "magt4m": <mean annual ground temperature value at 4 m depth (°C)>,
        "magt5m": <mean annual ground temperature value at 5 m depth (°C)>,
        "magtsurface": <mean annual ground temperature value at 0.01 m depth (°C)>,
        "permafrostbase": <depth of permafrost base (m)>,
        "permafrosttop": <depth of permafrost top (m)>,
        "talikthickness": <thickness of talik layer (m)>
      }
      ...
    }
    ...
  }
  ...
}

Query a specific time slice of GIPL model outputs (1 km spatial resolution) for ten variables for a single point specified by latitude and longitude. Summarize the results by computing the minimum, mean, and maximum values for each of the ten variables for the duration of the time slice. Statistics are computed for all models and scenarios.

Example: /permafrost/point/gipl/63.0628/-146.1627/2040/2050/mmm

Results from the above summary query will look like this:

{
  "gipl1kmmax": {...},
  gipl1kmmean": {...},
  gipl1kmmin": {...},
}
The above output is structured like this: 
{
    "gipl1kmmax": <max values for all ten variables across all models, scenarios, and years>,
    "gipl1kmmean": <mean values for all ten variables across all models, scenarios, and years>,
    "gipl1kmmin": <min values for all ten variables across all models, scenarios, and years>
}

CSV Export

To export the data from the above queries to a CSV file, append ?format=csv to the URL.

Example: /permafrost/point/gipl/63.0628/-146.1627?format=csv

Permafrost point query

Query permafrost data (e.g. extent, ground temperatures) for a single point specified by latitude and longitude.

Example: /permafrost/point/65.0628/-146.1627

Results from the query above will look like this:

{
  "gipl": {
    "1995": {
      "cruts31": {
        "historical": {
          "alt": 0.59,
          "magt": -2.2
        }
      }
    },
    ...
    "title": "Melvin et al. (2017) GIPL 2.0 Mean Annual Ground Temperature (°C) and Active Layer Thickness (m) Model Output"
  },
  "jorg": {
    "ice": "Moderate",
    "pfx": "Discontinuous",
    "title": "Jorgenson et al. (2008) Permafrost Extent and Ground Ice Volume"
  },
  "obu_magt": {
    "depth": "Top of Permafrost",
    "temp": 0.2,
    "title": "Obu et al. (2018) 2000-2016 Mean Annual Top of Permafrost Ground Temperature (°C)",
    "year": "2000-2016"
  },
  "obupfx": {
    "pfx": "Sporadic",
    "title": "Obu et al. (2018) Permafrost Extent"
  }
}
  • gipl: GIPL 2.0 Model Outputs
    • era: (str), Era name. "1995" refers to the historical baseline. Otherwise these are 30 year eras surrounding the following center dates, except last era is truncated at 2100: "2025" (2011 - 2040), "2050" (2036 - 2065), "2075" (2061 – 2090), "2095" (2086 – 2100).
      • model: (str), Climate model driving the GIPL model output.
        • scenario: (str), RCP Emissions Scenario driving the GIPL model output.
          • alt: (float), Active Layer Thickness (m).
          • magt: (float), Mean annual ground temperature (°C).
          • title: (str), Title describing these results.
  • jorg: Jorgenson et al. (2008) Permafrost Characteristics of Alaska
    • ice: (str), Estimated ground ice volume.
    • pfx: (str), Permafrost extent.
    • title: (str), Title describing these results.
  • obu_magt: Obu et al. (2018) Modeled Mean Annual Ground Temperature at Top of Permafrost
    • depth: (str), Top of permafrost (i.e. depth of active layer).
    • temp: (float), Mean annual ground temperature (°C).
    • title: (str), Title describing these results.
    • year: (str), Date range of model output.
  • jorg: Obu et al. (2018) Modeled Permafrost Extent
    • pfx: (str), Permafrost extent.
    • title: (str), Title describing these results.
