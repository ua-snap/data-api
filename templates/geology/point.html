Geology Point Query

Query the USGS Geologic Map of Alaska at a single point specified by latitude and longitude.

Example: /geology/point/69.0628/-144.1627

Results from the query above will look like this:

{
  "age": "Cambrian and Proterozoic",
  "name": "Wickersham and Neruokpuk units",
  "title": "USGS Geologic Map of Alaska"
}
  • title: (str), Title describing these results.
  • name: (str), Geologic unit name.
  • age: (str), Geologic unit age.
