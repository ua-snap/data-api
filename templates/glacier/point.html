Data API

Glacier Point Query

Query the glaciology of Alaska for a single point specified by latitude and longitude.

Example: /glacier/point/60.606/-143.345

Results from the query above will look like this:

{
  "ewisc": true, 
  "lwisc": true,
  "now": {
    "glacname": "Steller Glacier", 
    "modern": true
  }, 
  "pmax": true, 
  "title": "Past and Present Glaciology of Alaska"
}
  • title: (str), Title describing these results.
  • ewisc: (bool), The Early Wisconsin Glaciation did/did not cover this point with ice.
  • lwisc: (bool), The Late Wisconsin Glaciation did/did not cover this point with ice.
  • pmax: (bool), The Pleistocene Maximum Glacial Extent did/did not cover this point with ice.
  • now:
    • modern: (bool), The contempoary glacial extent does/does not cover this point with ice.
    • glacname: (str), Name of glacier or ice field, if available.
