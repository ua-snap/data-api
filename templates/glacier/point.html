Glacier Point Query

Query the glaciology of Alaska for a single point specified by latitude and longitude.

Example: /glacier/point/60.606/-143.345

What the endpoint returns

Results from the query above will look like this:

{
  "ewisc": true, 
  "lwsisc": true, 
  "now": {
    "glacname": "Steller Glacier", 
    "modern": true
  }, 
  "pmax": true, 
  "title": "Past and Present Glaciology of Alaska"
}

  • title: Title of these results
