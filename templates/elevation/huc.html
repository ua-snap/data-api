Elevation HUC Aggregation

Query 1 km interpolations of the 30 m resolution ASTER GDEM data for a specific HUC and aggregate by mean.

A query will fetch the minimum, mean, and maximum elevations found within a single HUC.

Example: /elevation/huc/19010208

Results from the query above will look like this:

{
  "res": "1 kilometer",
  "title": "ASTER Global Digital Elevation Model",
  "units": "meters difference from sea level",
  "max": 2961.0,
  "mean": 822,
  "min": 0.0
}
  • title: (str), Title describing these results.
  • res: (str), Spatial resolution (pixel size) of the DEM.
  • units: (str), Units for elevation `z` value.
  • z: (int), Elevation value.

Source data

Data Layer Download Link Data Notes
The Terra Advanced Spaceborne Thermal Emission and Reflection Radiometer (ASTER) Global Digital Elevation Model (GDEM) Version 3 (ASTGTM) Download Tiles z = 0 indicates sea level. Negative values are expected and indicate depressions below sea level.
