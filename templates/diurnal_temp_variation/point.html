Mean Annual Diurnal Temperature Variation (°C) Point Query

The endpoints here provide access to mean annual diurnal temperature variation data.

Examples

Query the downscaled ERA-Interim historical average (1980-2009) min, mean, and max mean annual diurnal temperature variation (°C) for a point location:

Example: /diurnal_temp_variation/historical/point/65.0628/-146.1627

Query the projected RCP 8.5 mid-century (2040-2069) two-model min, mean, and max mean annual diurnal temperature variation (°C) for a point location:

Example: /diurnal_temp_variation/projected/point/65.0628/-146.1627

Query the historical and projected min, mean, and max mean annual diurnal temperature variation (°C) for a point location:

Example: /diurnal_temp_variation/hp/point/65.0628/-146.1627

Query the historical or projected min, mean, and max mean annual diurnal temperature variation (°C) for a point location with specific start and end years:

Example: /diurnal_temp_variation/projected/point/65.0628/-146.1627/2070/2099

Results from the queries above will look like this: 
{
  "historical": {
    "madtrmax": 7, 
    "madtrmean": 7, 
    "madtrmin": 6
  }
}
The above output is structured as such: 
{
  <era>: {
    "madtrmax": <the max of the mean annual diurnal temperature variation (°C) across all models and years used for the summary era time period>,
    "madtrmean": <the mean of the mean annual diurnal temperature variation (°C) across all models and years used for the summary era time period>,
    "madtrmin": <the min of the mean annual diurnal temperature variation (°C) across all models and years used for the summary era time period>
  },
  ...
}

Comprehensive queries

The following query will fetch a complete mean annual diurnal temperature variation (°C) set of data composed of the ERA-Interim historical baseline (1980-2009) and the entire projection range (2006-2100) for both available models, GFDL-CM3 and NCAR-CCSM4 (both RCP 8.5).

Example: /diurnal_temp_variation/all/point/65.0628/-146.1627

Results from the query above will look like this: 
{
  "ERA-Interim": {
    "1980": {
      "madtr": 7
    }, 
    "1981": {
      "madtr": 7
    }, 
    "1982": {
      "madtr": 7
    }, 
...
}
The above output is structured as such: 
{
  <model>: {
    <year>: {
      "madtr": <mean annual diurnal temperature variation (°C)>
    },
    ...
  }
  ...
}

CSV Export

To export the point data to a CSV file, append ?format=csv.

Example: /diurnal_temp_variation/all/point/65.0628/-146.1627?format=csv
