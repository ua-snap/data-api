The following links provide access to state specific metadata created by the agencies that did the delineation; however, they are only good through January of 2012.  After that date, primary stewardship of the WBD transitioned from NRCS to the USGS.  Metadata after January of 2012 is much more generalized and should be obtained directly within the download area’s ( https://nrcs.app.box.com/v/huc/ ) copied and extracted zipped file geodatabases, through ArcCatalog, under the Description tab when looking directly at the layer file. 

 

Disregard the Spatial Reference information in these individual files and only use the Spatial Reference of the data that was delivered via the Geospatial Data Gateway.  These metadata files are provided only as information of the delineation process by each state and also include steps made after merging the datasets in to a seamless national layer.
